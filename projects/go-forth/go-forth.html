Spring Constants of Pullback Toy Cars (2020)

Final poster

Description

This was my 2.671 go forth and measure project. I set out to determine if there were significant manufacturing tolerances in die cast pullback toy cars to determine if any of them were better than others. I used logger pro and video to determine the position of the toy cars and then worked backwards to find the spring constants of each of the toy cars.

Lessons Learned

I was remote this semester and did not have a lot of space to do this project. If I were to do this again, I would have set up a video tracking software. I also would have built a more robust car release system.
