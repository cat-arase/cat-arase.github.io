Navigation Circuit Board (2021)

Final circuit board

Description

This is a navigation circuit board designed for my remote electronics project. This board was created using Eagle and sent off to be manufactured in China. The board is designed for internal navigation and uses an Arduino Nano and Sparkfun IMU to tell the user where to go.

Lessons Learned

I realized how important planning ahead is for designing electronics. Due to shipping times, the board took roughly a month to manufacture, so any mistakes had to be left as they were.
